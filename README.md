# Harjoitustyo
Aineopintojen harjoitusty√∂  

-----------------------------------------------------------
  
Installing instructions:  
Clone the repository to your pc:  
git clone https://github.com/MMorko/Harjoitustyo

Install dependencies:  
Poetry install

-----------------------------------------------------------

How to use:  
Launch play.py in your code editor or in the terminal:  
cd Harjoitustyo  

python src/play.py  

After launching you can choose your play piece, x goes first:  
<img width="133" height="23" alt="kuva" src="https://github.com/user-attachments/assets/f7cabb96-f2f4-4af4-8fee-966bd79dc179" />  

Then you can choose the depth the AI will use, time increases with depth:  
(In testing the depth was 8)  
<img width="149" height="17" alt="kuva" src="https://github.com/user-attachments/assets/1864d859-39d2-46a6-8e70-1c68837b2280" />  

Then lastly you can give the AI a time limit for each turn:  
<img width="299" height="19" alt="kuva" src="https://github.com/user-attachments/assets/91267c7d-e896-4344-85d5-0fa752969515" />
